JavaScript Looping Structures

This page demonstrates the use of for, while, and do-while loops in JavaScript.

1. For Loop

for (let i = 0; i < 5; i++) {
    console.log("For Loop Iteration: " + i);
}

2. While Loop

let j = 0;
while (j < 5) {
    console.log("While Loop Iteration: " + j);
    j++;
}

3. Do-While Loop

let k = 0;
do {
    console.log("Do-While Loop Iteration: " + k);
    k++;
} while (k < 5);
